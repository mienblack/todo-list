## 💻 Projeto

O todo list é uma lista de tarefas desenvolvida em React você pode criar tarefas, marca-las como feitas e deleta-las. As tarefas ficam salvas no seu armazenamento local.

## 🚀 Tecnologias

- [ReactJs]
- [Javascript]
- [HTML]
- [CSS]
